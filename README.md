# LESSON 82

반응형 웹기반 온라인주문 프로젝트

## 📁 프로젝트 소개

크몽, 숨고와는 달리 전문성과 체계적이고 집중적인 레슨을 희망하는 일반인과 다양한 분야의 전문가 사이를 연결하는 사용자 친화적인 레슨 상품 중점 시스템 웹 사이트를 개발하였습니다.

## 👨‍👩‍👧‍👦 개발 기간 및 인원

- 2024.06.07 ~ 2024.07.19
- 4명

## 프로젝트 주요 기능

- 회원, 전문가로 구분된 사용자 시스템
- 레슨 검색 및 필터링 기능
- 토스 API를 통한 결제 기능
